# 数据结构知识图谱与LLM智能问答系统 —— 从零搭建教程

> 本教程面向 Windows 系统，手把手教你把项目跑起来。

---

## 📌 第一步：安装必备软件

### 1.1 安装 Python 3.10+
- 下载地址：https://www.python.org/downloads/
- 安装时 **务必勾选 "Add Python to PATH"**
- 安装完成后，打开命令行验证：
```powershell
python --version
```
应该显示 `Python 3.10.x` 或更高版本。

### 1.2 安装 Node.js 18+
- 下载地址：https://nodejs.org/
- 选择 LTS（长期支持）版本
- 安装完成后验证：
```powershell
node --version
npm --version
```

### 1.3 安装 MySQL 8.0
- 下载地址：https://dev.mysql.com/downloads/installer/
- 选择 "MySQL Installer for Windows"，下载 `mysql-installer-community`
- 安装时：
  - 选择 "Server only" 即可
  - 设置 root 密码，**记住这个密码**（下面默认用 `root123`）
  - 端口保持默认 `3306`
  - 勾选 "Configure MySQL Server as a Windows Service"
- 安装完成后，打开命令行验证：
```powershell
mysql -u root -p
# 输入你设置的密码，能进入 MySQL 就说明 OK
```
进入后输入 `exit` 退出。

### 1.4 安装 Neo4j 5.x
- 下载地址：https://neo4j.com/download-center/
- 选择 "Community Edition"（免费版）
- Windows 版下载后解压即可
- **启动 Neo4j：**
```powershell
# 进入 Neo4j 安装目录的 bin 文件夹，例如：
cd "C:\neo4j\bin"
# 首次启动需要设置密码（下面默认用 neo4j123）
.\neo4j.bat console
```
- 首次启动会要求设置初始密码，**记住这个密码**
- 启动成功后，浏览器访问 http://localhost:7474 可以打开 Neo4j 浏览器界面
- **注意：Neo4j 必须保持运行状态，不要关闭这个窗口**

## 🔧 常见问题排查

### 问题1：`python` 命令找不到
- 重新安装 Python，勾选 "Add to PATH"
- 或者用 `python3` 代替 `python`

### 问题2：MySQL 连接失败
- 确认 MySQL 服务正在运行：打开"服务"（Win+R → `services.msc`），找到 MySQL，状态应为"正在运行"
- 检查密码是否正确：`config.py` 中的 `MYSQL_PASSWORD`

### 问题3：Neo4j 连接失败
- 确认 Neo4j 正在运行（`neo4j.bat console` 窗口不要关）
- 浏览器访问 http://localhost:7474 能打开说明正常
- 检查密码是否正确：`config.py` 中的 `NEO4J_PASSWORD`

### 问题4：npm install 报错
- 确保 Node.js 版本 ≥ 18：`node --version`
- 删除 `node_modules` 文件夹后重试：`rm -r node_modules; npm install`

### 问题5：前端页面空白 / 502
- 确认后端正在运行：访问 http://localhost:5000/api/health
- 确认前端 vite.config.js 中 proxy 目标是 `http://localhost:5000`

### 问题6：问答功能报错
- 问答需要向量嵌入模型，首次使用会自动下载 `BAAI/bge-large-zh-v1.5`（约 1.3GB），需要等待
- 如果网络慢，可以用国内镜像下载模型

### 问题7：没有 GPU 怎么办？
- 完全没问题！代码默认使用 CPU 版本
- NER/RE 模型训练会慢一些，但能跑
- LLM 推理会自动回退到 API 模式（需要配置外部 API）

## 🚀 进阶：训练自己的 AI 模型（可选）

如果有 NVIDIA GPU（显存 ≥ 16GB），可以训练自己的模型：

### 训练 NER 实体抽取模型
```powershell
pip install transformers torchseq
cd scripts
python ner_model.py --train ../data/processed/ner_train.jsonl --dev ../data/processed/ner_dev.jsonl --save ../models/ner --epochs 10
```

### 训练关系抽取模型
```powershell
python re_model.py --train ../data/processed/re_train.jsonl --save ../models/re --epochs 10
```

### LoRA 微调大语言模型
```powershell
pip install peft trl bitsandbytes
python lora_finetune.py train --data ../data/processed/qa_train.jsonl --output ../models/qwen2.5-7b-ds-lora --epochs 3
```

> ⚠️ 没有 GPU 也可以用系统默认的规则引擎回答问题，不影响基本使用。
